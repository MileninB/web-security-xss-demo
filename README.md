# К.Т.2 - XSS Demo на Node.js + Express

Учебный проект по дисциплине «Безопасность веб-приложений».

## Цель
Продемонстрировать хранимую XSS-уязвимость и реализовать защиту от неё.

## Реализовано
- форма добавления комментариев;
- сохранение комментариев в массив;
- уязвимый вывод комментариев без экранирования;
- демонстрация XSS-атаки;
- защита через экранирование вывода;
- Content Security Policy (CSP) через Helmet;
- санитизация пользовательского ввода через DOMPurify.
